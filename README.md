# Rock Paper Scissors

This is a web-based implementation of the classic game **Rock Paper Scissors** [project](https://www.theodinproject.com/lessons/foundations-rock-paper-scissors) built as part of [The Odin Project's JavaScript Curriculum](https://www.theodinproject.com/). Users can play against the computer, making choices that determine the outcome of each round.

## Features

- Interactive gameplay with user input
-	Score tracking for multiple rounds
-	Simple and intuitive user interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Goals

- Practice JavaScript fundamentals and game logic
- Develop skills in DOM manipulation
- Experiment with CSS styling to enhance the visual design
